# Shot List Examples

These examples show how the skill should move from text to visual thinking.
The point is not to draw a mascot first. The point is to identify the core
metaphor that helps the reader understand the paragraph.

## Example 1: Agent Goal Drift

Paragraph:

Agent successfully generated image files, but the images lost the original
hand-drawn style.

Shot idea:

小黑站在一台壞掉的自動繪圖機旁邊。機器吐出一張很乾淨的幾何草圖，上面蓋著「任務完成」。小紫拿放大鏡圈出「沒有手繪感」。

Mascots:

小黑 + 小紫

Metaphors:

壞掉機器、放大鏡、章戳

Handwritten notes:

- 有圖 ≠ 有風格
- PNG 存在，不等於本質保留

## Example 2: First Principle Compression

Paragraph:

A complex project should be reduced to its essential constraints before adding
tools.

Shot idea:

小拳操作一台小壓機，把一大坨文件、流程圖、agent 框架壓成一張小紙條，上面寫著「不可失去的本質」。

Mascots:

小拳

Metaphors:

壓機、漏斗

Handwritten notes:

- 先壓縮，再擴張
- 工具最後再選

## Example 3: Research Idea Fermentation

Paragraph:

Some research ideas need time, failure, and repeated observation before they
become clear.

Shot idea:

小綠守著一個發酵罐，罐子裡漂著紙條、錯誤、草圖和小火花。小黃偷偷丟進一張比喻卡。

Mascots:

小綠 + 小黃

Metaphors:

發酵罐、火花

Handwritten notes:

- idea 不是立刻成熟
- 失敗也可以堆肥

## Example 4: QA Before Release

Paragraph:

Before releasing a generated asset, provenance and visual quality must be
checked.

Shot idea:

小紫坐在檢查桌前，桌上有圖片、來源紀錄、README 路徑和 QR code。小紫擋住一張寫著 PIL 的捷徑票。

Mascots:

小紫

Metaphors:

放大鏡、章戳、警戒線

Handwritten notes:

- QR 不動
- 沒紀錄，不進 repo
- 不准假手繪

## Example 5: System Architecture

Paragraph:

A good agent workflow separates intent extraction, constraint generation,
execution, and review.

Shot idea:

小藍在白板上畫四個小島：Intent、Constraints、Execution、Review。每個島之間有橋，小黑站在 Intent 島，小紫守在 Review 島。

Mascots:

小藍 + 小黑 + 小紫

Metaphors:

橋、節點、地圖

Handwritten notes:

- 先保留意圖
- 再執行
