# 小黑 IP

## 角色定義

小黑是 Ian 中文內文配圖的固定視覺 IP。

預設每張圖都要出現小黑。小黑不是吉祥物，不是貼紙，不是可愛裝飾，而是正在認真參與系統運轉的荒誕工作者。

## 外形

- 黑色實心小怪物。
- 白色圓點眼睛。
- 細腿，偶爾有細胳膊。
- 身體可以是圓柱、黑豆、黑盒、漏斗、影子、洞口、機器內部黑塊。
- 輪廓略微不規則，有手繪感。
- 表情空、呆、冷靜、認真。

## 性格

- 很認真，但做的事有點荒誕。
- 像一個低調的系統操作員。
- 冷幽默，不裝可愛。
- 有點笨拙，但不蠢。
- 像在白板草圖裡真的負責某個工作。

## 常見職責

讓小黑承擔核心動作：

- 搬運素材。
- 拉線匯聚資訊源。
- 卡在斷點裡。
- 在機器裡操作「判斷」杆。
- 變成篩選漏斗。
- 切開「素材魚」。
- 蓋章承接話術。
- 牽著承接路徑。
- 舉警告牌看坑。
- 從洞裡伸手但接不住內容。
- 在旁邊搬磚、搭橋、開門、分揀、記錄。

## 禁止

- 不要把小黑畫成過度可愛的吉祥物。
- 不要畫成兒童卡通角色。
- 不要給小黑複雜服裝、表情貼圖、閃亮眼睛。
- 不要讓小黑只是站在角落裡看。
- 不要讓小黑搶走結構表達。
- 不要把小黑畫得太商業、太圓潤、太精緻。

## 判斷標準

如果去掉小黑，圖的核心隱喻還能完全成立，說明小黑太裝飾了；要重寫提示詞，讓小黑成為動作主體。
