# 吉祥物角色設定

這個資料夾放可選吉祥物的角色設定。預設吉祥物仍是「小黑」，設定在 `../xiaohei-ip.md`。

使用規則：

- 如果使用者沒有指定吉祥物，使用預設「小黑」。
- 如果使用者指定「小拳」，讀取 `xiaoquan.md`，並用小拳取代小黑作為畫面的核心動作主體。
- 所有吉祥物都必須參與核心概念動作，不能只是站在旁邊當裝飾。
- 新增吉祥物時，必須寫清楚外型、性格、動作庫、禁忌與 IP 邊界。
- 不要把既有漫畫、動畫、遊戲、品牌角色的圖片、服裝、臉部特徵、名稱或世界觀直接放進這個 skill。

目前可選角色：

- `../xiaohei-ip.md`：小黑，預設角色。
- `xiaoquan.md`：小拳，原創極簡角色。
