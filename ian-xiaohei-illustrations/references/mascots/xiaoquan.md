# 小拳 IP

## 角色定義

小拳是這個 skill 的可選原創吉祥物。它不是任何既有漫畫、動畫、遊戲或品牌角色的複製、改作或致敬圖像，而是一個用「最小動作解決複雜問題」來承載內文配圖隱喻的極簡角色。

小拳適合用在需要表達「直接切入核心」、「拆掉阻塞」、「把複雜問題壓成一個可行動作」、「用最小力氣完成關鍵判斷」的文章段落。它不應該變成英雄海報、戰鬥漫畫主角或熱血角色，而要維持白紙草圖裡的冷靜、低調和反差感。

## 外型

- 極簡小角色，身體可以是圓頭短身、黑白線稿小人、圓角小方塊或簡單豆形。
- 表情冷靜、空白、認真，眼睛可以是小黑點或小短線。
- 可以有一隻略大的手套拳頭，或戴一只簡單工作手套。
- 身體比例保持小，拳頭只是概念工具，不要讓畫面變成肌肉展示。
- 線條維持黑色手繪、輕微抖動、乾淨白底。
- 可用少量紅色標示阻塞點，橙色標示小拳打通的路徑，藍色標示系統回饋。

## 性格

- 冷靜、直接、少話。
- 不是熱血英雄，而是安靜的問題拆解員。
- 不誇張、不炫耀、不勝利姿勢。
- 像是在白板草圖中，用一個很小的動作把卡住的系統推回可運作狀態。
- 有一點荒誕反差：問題很大，動作很小，但剛好打在關鍵位置。

## 常見職責

讓小拳承擔核心動作：

- 輕敲阻塞的節點，讓流程重新接通。
- 用小拳把一大團混亂壓成一張可執行清單。
- 把多餘分支敲掉，只留下主路徑。
- 對準「真正問題」的小紅點，做一次精準處理。
- 把卡在漏斗口的素材往下推。
- 把散亂紙張壓平，變成一條可讀路徑。
- 用拳頭當小印章，標記「已決定」、「可行動」、「先做這個」。
- 擋住無關噪音，讓核心訊號通過。
- 敲開一個小門，讓下一步露出來。

## 禁止

- 不要使用任何既有漫畫、動畫、遊戲或品牌角色的圖像、名稱、服裝、臉部特徵或世界觀。
- 不要畫黃色緊身衣、白披風、紅手套、光頭英雄、漫畫式戰鬥姿勢或熱血分鏡。
- 不要出現既有作品名稱、角色名、標誌、招式名或明確可辨識的致敬構圖。
- 不要讓小拳變成肌肉英雄、格鬥角色、超能力角色或商業吉祥物。
- 不要畫爆炸、暴力擊打人體、受傷角色或破壞性戰鬥場面。
- 不要讓小拳只是站在旁邊看；它必須承擔核心概念動作。

## 判斷標準

如果去掉小拳，圖的核心隱喻仍然完全成立，代表小拳太裝飾了；要重寫提示詞，讓小拳成為解決阻塞、收斂複雜度或推動流程的核心動作主體。
